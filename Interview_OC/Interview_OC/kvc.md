# KVC (Key-Value Coding) 示例演示

## 概述
本示例展示了 Objective-C 中 KVC 的多种用法，包括字典模型互转、动态赋值、配合 KVO、操作私有成员变量等。

## 核心功能演示

### 1. 字典与模型互转
- **字典转模型**: 使用 `setValuesForKeysWithDictionary:` 方法
- **模型转字典**: 使用 `dictionaryWithValuesForKeys:` 方法
- **安全处理**: 实现 `setValue:forUndefinedKey:` 方法处理未知键

### 2. 动态赋值
- **动态设置属性**: 根据运行时信息设置对象属性
- **集合操作**: 使用 `mutableArrayValueForKey:` 操作集合属性
- **批量操作**: 对对象数组进行属性提取和计算

### 3. 操作私有成员变量
- **访问私有变量**: 使用 KVC 访问类中声明的私有成员变量
- **修改私有变量**: 即使没有 setter/getter 方法也能修改私有变量
- **异常处理**: 安全处理不存在的私有变量访问

### 4. 配合 KVO 使用
- **KVO 观察**: 添加对属性的观察
- **KVC 触发**: 使用 KVC 修改属性值触发 KVO 通知
- **安全移除**: 在 dealloc 中安全移除观察者

### 5. 高级用法
- **集合运算符**: 使用 `@sum`、`@avg`、`@max`、`@min` 进行数值计算
- **键路径操作**: 使用键路径访问嵌套属性
- **批量获取**: 从对象数组中提取特定属性值

## 代码结构

### Person 类
- **Person.h**: 类声明，包含公有属性和私有成员变量
- **Person.m**: 类实现，包含 KVC 安全处理方法

### ViewController 类
- **ViewController.m**: 包含完整的 KVC 演示代码，分为多个演示方法

## KVC 核心概念

### 基本方法
- `setValue:forKey:`: 为指定键设置值
- `valueForKey:`: 获取指定键的值
- `setValue:forKeyPath:`: 为键路径设置值
- `valueForKeyPath:`: 获取键路径的值

### 集合运算符
- `@count`: 返回集合中对象数量
- `@sum`: 返回集合中指定属性的总和
- `@avg`: 返回集合中指定属性的平均值
- `@max`: 返回集合中指定属性的最大值
- `@min`: 返回集合中指定属性的最小值

### 安全注意事项
1. **键名正确性**: 确保键名与属性名一致
2. **类型匹配**: 设置的值类型需要与属性类型兼容
3. **异常处理**: 使用 try-catch 处理可能的异常
4. **KVO 安全**: 确保在适当的时候添加和移除观察者

## 实际应用场景

### 1. JSON 解析
使用 KVC 快速将 JSON 字典转换为模型对象

### 2. 动态表单
根据配置动态设置和获取表单字段值

### 3. 数据绑定
配合 KVO 实现 UI 与数据的自动同步

### 4. 反射编程
在运行时动态操作对象属性和方法

## 注意事项
1. KVC 会绕过属性修饰符（如 readonly）
2. 访问不存在的键会触发异常
3. 性能考虑：KVC 比直接访问属性稍慢
4. 类型安全：编译器无法检查 KVC 的类型安全
